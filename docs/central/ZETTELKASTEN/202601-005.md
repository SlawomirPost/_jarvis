## Fakt

Cztery poziomy autonomii agenta AI:

| Poziom | Nazwa | Model | Opis |
|--------|-------|-------|------|
| **L1** | Human-in-the-loop | `require_approval: True` | Agent prosi o pozwolenie na każdą zmianę |
| **L2** | Human-on-the-loop | Whitelist + threshold | Auto dla safe, pyta dla risky |
| **L3** | Autonomous | Full allowlist | Agent działa sam, human monitoruje via logs |
| **L4** | Self-improver | Unrestricted + telemetry | Agent modyfikuje własny kod |

Mapowanie na fazy Jarvis:
- Faza 1-2: L1 (budowanie zaufania)
- Faza 2-3: L2 (sprawdzone operacje auto)
- Faza 3: L3 (24/7 z monitoringiem)
- Faza 4: L4 (self-improvement)
