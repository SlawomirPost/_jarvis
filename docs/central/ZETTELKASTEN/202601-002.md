## Fakt

Tool Runner (beta) to nowy pattern w API Anthropic który automatycznie zarządza całym cyklem tool use:
- Iteracje aż do end_turn (bez manualnej pętli)
- Parallel tool calls
- Error handling
- Message history management

Eliminuje boilerplate ręcznego zarządzania `tool_use` / `tool_result` messages.

## Kontekst

Dla Fazy 2 Jarvis (Telegram, Gmail, Calendar) Tool Runner jest idealny — pozwala Claude automatycznie wywoływać wiele narzędzi w odpowiedzi na jedno polecenie użytkownika.

Przykład: "Wyślij email do Kowalskiego i stwórz spotkanie" → Claude sam zdecyduje o kolejności/równoległości.
