## Fakt

FastMCP to wysokopoziomowy framework Python do tworzenia serwerów MCP. Drastycznie upraszcza implementację — dekorator `@mcp.tool()` automatycznie generuje schematy JSON i obsługuje komunikację JSON-RPC.

Kluczowe cechy:
- Transport STDIO dla Claude Code (najszybszy)
- Automatyczna walidacja via Pydantic
- Asyncio-native

## Kontekst

Dla Fazy 1 Jarvis (local-first, $0 kosztów) FastMCP jest optymalnym wyborem — pozwala zbudować `jarvis-notes` server z CRUD na markdown w kilkadziesiąt linii kodu.
